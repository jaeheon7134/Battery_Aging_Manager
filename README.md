# Battery_Aging_Manager

04-21

로그 분석 조건 필터(온도/구간/사이클) 및 폴더 로드 개선

폴더 선택 시 하위 폴더까지 재귀 탐색하여 CSV 일괄 로드
조건 적용 ON/OFF에 따라 조건 그룹 활성/비활성 및 즉시 반영
온도 조건 추가: 각 CSV 첫 battery_temp 기준으로 슬라이더 값과 1도 미만 파일만 표시
구간 선택 추가: Mode 컬럼에서 충전/방전/전체 필터링
구간 체크박스 동작 정리: 전체/충전/방전 배타 선택 및 기본값 보정
사이클 조건 추가: repeat time이 콤보박스 선택값과 일치하는 파일만 표시
리스트/그래프 필터 동작 일치화(조건 불일치 항목 자동 체크 해제)
CSV 한글 값 파싱 이슈 수정(시스템 인코딩 적용)
성능 최적화:
파일 메타(첫 온도, 사이클 집합) 캐시 도입
폴더 로드 중 반복 필터링 제거 후 일괄 갱신
필터 중 리스트 시그널 블로킹으로 불필요한 재연산 감소
슬라이더 드래그 중 라벨만 갱신, release 시 필터/그래프 1회 갱신
